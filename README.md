# Termometr – projekt IoT

## Opis projektu
Projekt „Termometr” to urządzenie i aplikacja stworzone w ramach własnej nauki oraz eksperymentów z elektroniką i programowaniem. Celem projektu było zbudowanie prostego systemu pomiaru temperatury, który odczytuje dane z czujnika, prezentuje je w przystępnej formie i może być rozbudowywany o kolejne funkcje.

Projekt został wykonany hobbystycznie, a cały kod oraz dokumentacja techniczna znajdują się w tym repozytorium.

## Funkcjonalności
- Odczyt temperatury z czujnika
- Przesyłanie i prezentacja danych
- Podstawowa logika aplikacji do obsługi pomiarów

## Technologie
- ESP32
- BMP280
- Blynk
